# Digital Grievance Redressal System

## 🚩 Problem Statement

In many educational institutions, grievance handling is manual, unstructured, and lacks transparency. Students often lack a proper system to track complaint status, leading to delays and inefficiency.

## 💡 Proposed Solution

The Digital Grievance Redressal System is a web-based application designed to provide a structured and transparent platform for managing student complaints. It enables students to submit grievances online and track their resolution status in real time.

## 🚀 Key Features

- User Authentication (Student & Admin roles)
- Complaint Submission with category selection
- Status Tracking (Pending / In Progress / Resolved)
- Admin Dashboard for grievance management
- Role-Based Access Control

## 🛠 Tech Stack

- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- Authentication: JWT
