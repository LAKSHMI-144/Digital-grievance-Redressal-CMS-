# Digital-grievance-Redressal-CMS-
A role-based Digital Grievance Redressal System built using the MERN stack to enable transparent and efficient complaint management within educational institutions.
🚩 Problem Statement
In many educational institutions, grievance handling is manual, unstructured, and lacks transparency. Students often do not have a proper system to track the status of their complaints, leading to delays and inefficiency.
💡 Solution
This project provides a web-based Digital Grievance Redressal System that allows students to submit complaints online and track their status in real time. Administrators can manage, update, and resolve grievances efficiently through a centralized dashboard.
🚀 Features
User Authentication (Student & Admin roles)
Complaint submission with category selection
Status tracking (Pending / In Progress / Resolved)
Admin dashboard for grievance management
Role-based access control
Responsive UI
🛠 Tech Stack
Frontend: React
Backend: Node.js + Express
Database: MongoDB
Authentication: JWT
